Current CLI Behavior:
Runs an interactive loop using Rich:
    * Commands:
        - help 
        - test
        - parse-trs 
            -- Prompts for TRS CSV filename and calls runParseTRS()
        - exit, quit, q 
            -- exits the CLI
    * Styled output using Rich (Prompt, Panel, Rule)

Tech Stack:
* Python
* Rich for CLI styling
* uv for dependency management
* Git for version control and remote saving

Key Scripts & Their Roles

main.py:
- CLI entry point, interactive loop, command routing.
- Calls parsing and formatting functions.

common.py:
- Shared utility functions (e.g., getRootFolder, getColumnHeaders).

parse_trs.py:
- Reads TRS files, profiles columns, analyzes patterns (especially ~UDCALL).
- Returns a summary dictionary for Rich formatting.

parse_pnt.py:
- Reads PNT exports, profiles columns, handles UDC mapping.
- Returns a summary dictionary for Rich formatting.

rich_formatting.py:
- Contains functions like display_trs_summary and display_pnt_summary.
- Uses Rich to format output: panels, tables, dynamic column layouts.

Recent Changes & Improvements:

TRS Analysis
    Profiles columns for:
        - Total rows
        - Headers 
        - Non-empty counts
        - Percentages
        - Unique counts
        - Empty columns 
        - Missing values

    Special focus on ~UDCALL table:
        - Duplicate descriptions and associated ENTRYs.
        - Prefix grouping for subsystem analysis.
        - Mapping structure {ENTRY: DESC} for tag reference.

PNT Analysis:
    Profiles columns for:
        - Total rows
        - Fully populated columns
        - Missing values 
        - Empty categories
        - Unique UDC 
        - Description counts

    Improved missing value logic:
        - Converts empty strings and whitespace to pd.NA for accurate missing value detection.

    Separates columns into:
        - Fully populated, partially missing, and fully empty.

    Adds logic to find Unaccounted Columns:
        - Compares original headers to all analyzed lists to find any columns not categorized.

Rich CLI Formatting
    - Modular functions for displaying summaries.
    - Dynamic column distribution for long lists (e.g., fully populated columns, empty categories).
    - Displays counts for each section.
    - Uses Rich tables and panels for compact, readable output.
